# HealthCare-Analysis-Dashboard

## 📊 Overview

This project presents a **Power BI Healthcare Dashboard** that analyzes hospital financial and operational data, including billing, treatment costs, insurance coverage, and departmental performance.

The dashboard provides **actionable insights** to improve decision-making in healthcare management.

---

## 📊 Key Metrics

| Metric                  | Value Range    |
| ----------------------- | -------------- |
| 💰 Total Billing Amount | £1M – £3M      |
| 📈 Average Billing      | £674 – £709    |
| 💊 Medication Cost      | £212K – £546K  |
| 🏥 Treatment Cost       | £1M – £3M      |
| 🛡 Insurance Coverage   | £881K – £2.23M |
| 💸 Out-of-Pocket        | £504K – £1.13M |
| 🛏 Room Charges         | £141K – £180K  |

---

## 🧠 Insights

### 💉 Procedure-wise Billing

* **Top Revenue Generators:**

  * X-Ray
  * CT Scan
  * MRI Scan

* **Lower Contribution:**

  * Ultrasound
  * Blood Tests

---

### 🏥 Department-wise Performance

* **Orthopedics** leads with ~40% contribution

**Other departments:**

* Cardiology
* General Surgery
* Neurology
* Pediatrics

---

### 🩺 Diagnosis & Services

Key conditions analyzed:

* Asthma
* Hypertension
* Fracture
* Appendicitis
* Migraine

📌 Service types vary significantly in cost across diagnoses.

---

### 💰 Financial Insights

* Insurance covers a major portion of expenses
* Patients still bear noticeable out-of-pocket costs
* Balanced distribution between treatment & medication
